PcDuinoV2-Debian
================

Scripts to create an Image of Debian or only kernel for PcDuino V2

Installation steps
------------------

```shell
sudo apt-get -y install git
cd ~
git clone https://github.com/bright-things/PcDuinoV2-Debian
chmod +x ./PcDuinoV2-Debian/build.sh
cd ./PcDuinoV2-Debian
sudo ./build.sh
```
